# Java example using singleton pattern, single responsibility  principle and multithreaded-safe

### Tech stack:
* Java 21
* Lombok
* Gradle

## Requirements
* Java 21 (Any JDK)

## Build setup
```bash
# clone repository
$ git clone https://github.com/hermippus/java-singleton-spr-example

# build with Gradle
$ ./gradlew build

# run with jar file
$ java -jar build/libs/study-1.0.jar
```
